# Glossary

영어와 대응되는 한글목록입니다. 이해하기 쉬운 수준에서 최대한 한글로 표기함을 지향하였습니다. 다만, 한글에 대응되는 개념이 오히려 설명을 어렵게한다고 생각되거나(e.g. mapping: 사상), 영단어가 그 자체로 고유명사화 된 경우(e.g. agent: 에이전트)는 영어원문 그대로 표기하는 것을 원칙으로 합니다.
